추정 분할점수 시뮬레이터 사용 매뉴얼

개요

이 시뮬레이터는 성취평가제 추정 분할점수를 산출합니다.
각 문항 범주(배점합)에 대해 성취수준 경계별 예상 정답률을 입력하면,
분할점수 = Σ(배점합 × 예상정답률 / 100) 공식으로 자동 계산됩니다.

① 과목 유형 선택

  • 공통과목: A/B · B/C · C/D · D/E · E/I — 5개 경계 (고교학점제 이수·미이수 포함)
  • 선택과목: A/B · B/C · C/D · D/E — 4개 경계 (E/I 없음)

주의: 유형 변경 시 입력 데이터와 계산 결과가 초기화됩니다.

② 정기시험(중간·기말) 입력

배점합 입력

  • 선택형(쉬움·보통·어려움), 서답형(쉬움·보통·어려움) 6개 범주의 배점 합계를 입력합니다.

예상 정답률 입력

  • A/B ~ D/E 드롭다운에서 5% 단위(5, 10, …, 100)로 선택합니다.
  • E/I 공통과목 이수·미이수 경계는 1% 단위 정수로 직접 입력합니다.
  • 뒤쪽 경계(B/C, C/D, D/E)의 선택지는 앞 경계 값 이하로 자동 제한됩니다.

오류 표시

  • 주의 표시 경계 역전 또는 난이도 역전 — 해당 칸을 수정하세요.
  • 급간 표시 5% 급간 오류(E/I 칸에 소수 입력 등).

③ 라운드 운영

  • 나이스(NEIS) 권장: 2라운드 이상 실시합니다.
  • 교사별로 각 라운드에 입력값을 넣으면 개인별 추정 분할점수를 확인할 수 있습니다.
  • + 라운드 추가 버튼으로 라운드를 늘리고, 라운드 탭을 클릭해 전환합니다.
  • 각 라운드의 저장 버튼으로 해당 라운드만 엑셀로 저장할 수 있습니다.
  • 모든 라운드의 평균값이 최종 추정 분할점수가 됩니다.

④ 수행평가 입력

  • + 영역 추가로 수행평가 영역을 추가하고 영역명과 배점합을 입력합니다.
  • 정답률은 5% 단위 제한 없이 자유롭게 입력합니다.
  • 경계 역전(앞 경계보다 높은 값)만 오류로 표시됩니다.

⑤ 학기말 합산

  • 중간고사·기말고사·수행평가 각각의 반영 가중치(%)를 입력합니다.
  • 합계가 100%가 되어야 최종 계산이 가능합니다.
  • 학기말 분할점수 = 중간 × 가중치 + 기말 × 가중치 + 수행 × 가중치

⑥ 데이터 저장 · 복원

JSON 저장 / JSON 불러오기

  • 현재까지 입력한 모든 라운드 데이터를 .json 파일로 저장합니다.
  • 저장한 파일을 불러오면 이전 입력 상태가 완전히 복원됩니다.
  • 협의 전·후 데이터를 별도 파일로 관리하는 데 유용합니다.

엑셀 저장

  • 각 탭의 엑셀 저장 버튼으로 해당 평가의 전체 라운드를 엑셀로 내보냅니다.
  • 학기말 합산 탭의 엑셀 다운로드로 전체 시트를 한 번에 저장합니다.

⑦ 결과 확인

  • 계산하기 버튼을 누르면 라운드별·평균 분할점수가 카드와 표로 표시됩니다.
  • 시각화 바로 성취수준별 점수 구간을 한눈에 확인할 수 있습니다.
  • 2개 이상 라운드가 있으면 라운드별 비교 표도 함께 표시됩니다.
